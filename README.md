# 🤖 Instagram Account Creator Bot

Automated Telegram bot for creating Instagram accounts with temporary email services, captcha solving, and 2FA setup.

## ✨ Features

- ✅ **5 Temporary Email Providers** with intelligent rotation
- ✅ **Free OCR Captcha Solving** (Tesseract) with 2Captcha fallback
- ✅ **Automatic 2FA Setup** with secret key extraction
- ✅ **Real Instagram API** integration
- ✅ **Complete Account Credentials** delivery
- ✅ **Rate Limiting** and user management
- ✅ **Render Deployment** ready

## 📋 Email Providers

1. **TempMail** (Primary) - 99% reliability
2. **GuerrillaMail** (Backup) - 95% reliability
3. **Mailinator** (Secondary) - 90% reliability
4. **10MinuteMail** (Alternative) - 92% reliability
5. **TempEmailAddress** (Last Resort) - 88% reliability
